DEADZONE / QUARANTINE Simulator

이 프로젝트는 사용자가 제공한 두 개의 원문 문서를 토대로 구성한 정적 웹 기반 시뮬레이터입니다.
실행 파일은 index.html 하나이며, 별도 빌드 도구 없이 로컬에서 열거나 GitHub Pages에 업로드해 사용할 수 있습니다.

핵심 구조
1. sources/
   - 사용자가 제공한 원문 전체를 그대로 보관합니다.
2. data/world.js
   - 시뮬레이터가 읽는 통합 데이터 레이어입니다.
   - 시나리오를 DEADZONE / QUARANTINE으로 분리하여 도시명·시대 차이 충돌을 막았습니다.
3. js/app.js
   - 판정, 이동, 아이템, 감염, 로그, 저장/불러오기 기능을 담당합니다.
4. css/styles.css
   - 정적 UI 스타일입니다.

설명
- 문서에 없는 룸 단위 세부 구조는 억지로 확장하지 않았습니다.
- 대신 문서에 명시된 조사 위치와 행동 포인트를 직접 행동 버튼으로 만들었습니다.
- 수치가 비어 있는 부분은 2D6 판정형 규칙으로 보강했습니다.

실행 방법
- index.html을 브라우저에서 바로 열기
또는
- GitHub repository root에 업로드 후 GitHub Pages 활성화

주의
- 원문 전체가 구조화 JSON으로 100% 완전 파싱된 것은 아닙니다.
- 다만 원문 자체는 sources 폴더에 모두 포함되어 있으며, 시뮬레이터에서 실제 사용되는 핵심 장소/시스템/NPC/레시피는 통합 데이터에 등록되어 있습니다.
