UPDATE NOTES

사용자 요청 반영 사항
1. 처음 제공한 자료를 프로젝트에 모두 등록
   - sources/deadzone_source.txt
   - sources/quarantine_zone_source.txt
2. 부족한 내용 보강
   - 통합 판정식 추가: 2D6 + 능력치 >= 목표치
   - 포만도/수분/턴 진행 로직 추가
   - 세이브 내보내기 / 불러오기 추가
   - 제작 UI 및 레시피 소비 로직 추가
3. 충돌 방지 설계
   - DEADZONE과 QUARANTINE은 서로 다른 도시/시간축으로 보여 충돌 가능성이 있어 시나리오 분리 처리
   - 공통 시스템만 상단 엔진 레이어로 통합

추가된 내부 규칙
- 하루는 3개 행동마다 1일 진행
- 행동마다 포만도/수분 -1
- 포만도 또는 수분이 0이면 HP 감소
- 감염 판정은 QUARANTINE 문서 기준을 단순화해 자동 적용

향후 확장 포인트
- 모든 조사 위치 32개 전량 JSON 세분화
- NPC 합류/관계도 시스템
- 엔딩 판정 분기
- 지도 시각화
- 멀티 파티 지원
